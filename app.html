rundyfi.py¶

Create DYFI products for a given event ID

usage: rundyfi.py [-h] [--push] [--redo] [--configfile CONFIGFILE] evid

Positional Arguments¶

evid

Event ID

Named Arguments¶

--push

Push event products to PDL

Default: False

--redo

Redo all products even if they exist

Default: False

--configfile

Specify config file

Default: “./config.yml”

makemovie.py¶

Create movie frames for a given event

usage: app/makemovie.py [-h] [--tmax [TMAX]] [--framelength [FRAMELENGTH]]
                        [--type [TYPE]]
                        evid

Positional Arguments¶

evid

Event ID

Named Arguments¶

--tmax

Length of movie in minutes (default 60)

Default: 60

--framelength

Time of each frame, in seconds (default 60)

Default: 60

--type

Type of aggregation (default geo_10km)

Default: “geo_10km”

makescreenshot.py¶

Create static image .png files for a given event

usage: app/makescreenshot.py [-h] [--configfile CONFIGFILE] input [output]

Positional Arguments¶

input

GeoJSON file

output

output (default is screenshot.png)

Default: “screenshot.png”

Named Arguments¶

--configfile

Specify config file

Default: “./config.yml”
